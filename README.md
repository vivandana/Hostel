# Hostel Management App

This is a project for managing hostel notices, room allocations, fees, and more using Kotlin and Firebase.

## Features
- Admin & Student Login
- Notice Board
- Room Allocation
- Fees Tracker

## Tech Stack
- Kotlin
- Jetpack Compose
