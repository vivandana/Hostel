# Hostel Management App

This is a project for managing hostel room allocations, fees, Student registration and more using Kotlin and XML.

## Features
- Notification & Mess Meal Routine
- Student Registration
- Room Allocation
- Fees Tracker

## Tech Stack
- Kotlin
- Jetpack Compose
